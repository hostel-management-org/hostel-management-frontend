# Hostel Management System – Frontend

This is the **React frontend** for the Hostel Management System.  
It allows students to view and manage hostel allocations, rooms, and other details. //project 

---

🚀 **Live Frontend URL (Vercel)**  
👉 [https://hostel-management-frontend-beta.vercel.app/](https://hostel-management-frontend-beta.vercel.app/)  

---

⚠️ **Important Usage Note**  
This frontend depends on the backend service.

👉 Before using the frontend:  
1. Open the backend URL once:  
   [https://hostel-management-backend-rc9d.onrender.com/api/students](https://hostel-management-backend-rc9d.onrender.com/api/students)  
2. Wait for the backend to start (Render Free Plan may take 30–60 seconds)  
3. Then open the frontend URL  

> If the backend is not running, data may not load correctly.
